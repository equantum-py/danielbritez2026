# Imágenes necesarias

No usar fotografías falsas ni rostros generados por IA. Hasta recibir material real, el sitio utiliza una composición técnica abstracta creada con CSS y SVG.

Agregar imágenes reales de Daniel Brítez, optimizadas en formato WebP o AVIF:

1. `daniel-britez-principal.webp`: retrato vertical profesional de Daniel Brítez.
2. `daniel-britez-moto-taller.webp`: fotografía real de Daniel trabajando con una motocicleta.
3. `daniel-britez-repuestos.webp`: fotografía de Daniel en el área comercial, atendiendo clientes o trabajando con repuestos.
4. `daniel-britez-diagnostico.webp`: fotografía real de herramientas, medición, diagnóstico o revisión técnica.

## Recomendaciones

- Exportar cada imagen con un ancho máximo aproximado de 1200 px.
- Utilizar buena iluminación.
- Mantener el fondo limpio y ordenado.
- Evitar marcas visibles que no estén autorizadas.
- No usar fotografías de stock para representar a Daniel.
- No usar rostros generados por inteligencia artificial.
- Mantener una apariencia natural y profesional.
- Comprimir las imágenes sin perder calidad visual.
- Utilizar nombres de archivo exactamente iguales a los indicados.
